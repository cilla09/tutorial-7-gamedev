# Tutorial 7 Game Development
### Priscilla Natanael Surjanto/2306152153

Dalam tutorial ini, saya mengembangkan sebuah proyek game 3D yang mencakup pembuatan player, world/level, serta berbagai mekanik dasar seperti pergerakan menggunakan tombol WASD, lompat, dan interaksi. Selain itu, saya juga mengimplementasikan fitur switch yang dapat digunakan untuk menyalakan dan mematikan lampu di dalam level. Untuk latihan mandiri, saya merancang sebuah level dengan obstacle yang harus dilewati pemain untuk mencapai AreaTrigger sebagai tujuan akhir. Saya juga menambahkan sistem pickup item dan inventory yang menampilkan ikon serta nama item, di mana inventory dapat ditampilkan atau disembunyikan menggunakan tombol Tab. Untuk memperkaya mekanik permainan, player juga dilengkapi dengan kemampuan sprint dengan menahan tombol Shift untuk bergerak lebih cepat, serta crouch dengan menahan tombol C untuk memperlambat gerakan dan menurunkan sudut pandang.
